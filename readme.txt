Integrate the plugin to your website and start receiving payments in crypto.

== Description ==

The ivendPay Wordpress plugin allows you to integrate a payment page directly into your website for WooCommerce plugin.
Your customers will be able to use various crypto currencies to pay for the products and services you offer them.
The installation of the plugin is easy and intuitive process.

For detailed information and legal text please see https://ivendpay.com/

== Installation ==

= Prerequisites =

- You must be using WooCommerce 3.0 or newer
- You must enable WordPress pretty permalinks in Settings > Permalinks.

= Installation Instructions =

- Upload the plugin files to WordPress.
- Activate the plugin through the Plugins screen in WordPress.
- Go to WooCommerce -> Settings -> Payments -> ivendPay (click Manage/Set up) and fill out required settings or click Settings link on plugin page in section ivendPay for WooCommerce
